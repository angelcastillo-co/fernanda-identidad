CONCEPTO VISUAL

El Reto

Crear una identidad que se vea elegante sin ser delicada, femenina sin ser cliché, y comercial sin ser genérica. Una marca que tenga presencia real en redes y producto.

La Solución

Reinterpretación de la flor de lis con formas arquitectónicas, contraste alto, y sistema visual con riqueza. No minimalismo vacío. Identidad reconocible a simple vista.

PROPUESTA SELECCIONADA

Orgánica Arquitectónica: Crecimiento + Estructura

LOGO PRINCIPAL
PNG · Fondo transparente

Flor de lis geometrizada con estructura fuerte. Combina elegancia con peso visual. El isotipo funciona de forma independiente y se adapta a todos los tamaños sin perder carácter. Perfecto equilibrio entre sofisticación y presencia comercial.

ÁREA DE PROTECCIÓN

El logo debe tener un espacio mínimo de protección equivalente al ancho de la letra "A" de FERNANDA en todos sus lados. Este espacio debe estar libre de texto, imágenes u otros elementos gráficos.

Medida de referencia:

  • Alto contraste entre elementos

  • Degradados suaves
  • Colores pastel o apagados
  • Uso tímido del vino
  • Fondos grises intermedios

TIPOS DE ARCHIVO Y USOS

.PDF (Portable Document)

Universal, conserva vectores

→ Impresión digital, documentos, intercambio
.SVG (Scalable Vector)

Vector para web, escalable sin pérdida

→ Sitio web, apps, desarrollo digital
.PNG (Portable Network Graphics)

Imagen con transparencia, alta resolución

→ Redes sociales, presentaciones, email
.PSD (Photoshop)

Templates editables con capas

→ Mockups, contenido redes, fotografía

GUÍA DE USO RÁPIDO

Para publicar en Instagram
→ Usar .PNG (1080×1080px o superior)
Para enviar a imprenta
→ Usar .PDF (vectorial, CMYK)
Para sitio web
→ Usar .SVG (escalable, ligero)
Para presentación PowerPoint
→ Usar .PNG (fondo transparente, 300 DPI)
